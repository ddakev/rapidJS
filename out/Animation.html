Animation

Describes an animation for a game object

Constructor

new Animation(name, image, rows, cols, duration, repeat)

Constructor with parameters name, spritesheet, number of rows and columns, duration, and whether to repeat
Parameters:
Name Type Description
name string name of animation
image Image sprite sheet for animation
rows number number of rows in the spritesheet (defaults to 1)
cols number number of cols in the spritesheet (defaults to 1)
duration number duration of animation in milliseconds
repeat boolean whether the animation should repeat after it ends
